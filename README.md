# Bank Account Simulator

A beginner-level C++ program that simulates a simple banking system. Users can create accounts with unique account numbers, deposit money, withdraw money, and check balances. This project is perfect for practicing **functions, arrays, loops, and basic input/output**.

## Features
- Create accounts with unique account numbers  
- Deposit money  
- Withdraw money  
- Check account balances  
- Menu-driven interface  
- Error messages for duplicate accounts or insufficient balance
